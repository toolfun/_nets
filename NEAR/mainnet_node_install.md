### Update and install developer tools
```
sudo apt update
sudo apt install git make binutils-dev libcurl4-openssl-dev zlib1g-dev libdw-dev libiberty-dev cmake gcc g++ python docker.io protobuf-compiler libssl-dev pkg-config clang llvm cargo awscli -y
```

### If python was not installed
```
sudo apt install python-is-python3
```

### Rust installation
```
sudo curl https://sh.rustup.rs -sSf | sh -s -- -y
source $HOME/.cargo/env
rustup update
```

### Clone and compile binary
```
cd
git clone https://github.com/near/nearcore
cd nearcore
git fetch origin --tags
git checkout tags/1.36.5 -b mynode
make release
```
